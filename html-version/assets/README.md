# Assets

Copie os seguintes arquivos para esta pasta antes de fazer upload na hospedagem:

- `cuidame-logo.png` → de `src/assets/cuidame-logo.png`
- `hero-mockup.png` → de `src/assets/hero-mockup.png`
- `love-section-family.jpg` → de `src/assets/love-section-family.jpg`
- `favicon.png` → de `public/favicon.png`
- `og-image.png` → de `public/og-image.png`

## Estrutura para upload

```
html-version/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── assets/
    ├── cuidame-logo.png
    ├── hero-mockup.png
    ├── love-section-family.jpg
    ├── favicon.png
    └── og-image.png
```
